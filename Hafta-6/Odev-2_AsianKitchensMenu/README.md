# Asian Kitchen Menu

Asian Kitchen Menu, çeşitli Asya mutfaklarını ve yemeklerini listeleyen basit bir web uygulamasıdır. Kullanıcılar menüdeki yemekleri kategori bazında filtreleyebilir.

## Özellikler

- Menü öğelerini kategoriye göre filtreleyebilme (Korea, Japan, China).
- Menü öğeleri için başlık, fiyat, görsel ve açıklama görüntüleme.
- Responsive tasarım sayesinde farklı ekran boyutlarına uyumlu.
- Bootstrap ve Google Fonts kullanımı ile görsel olarak estetik bir tasarım.

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Google Fonts (`Great Vibes`, `Zilla Slab`)

## Kullanım

1. Projeyi bilgisayarına klonlayın veya ZIP olarak indirin.
2. `index.html` dosyasını herhangi bir tarayıcıda açın.
3. Menü öğelerini görmek ve filtrelemek için kategori butonlarına tıklayın.

## Menü Kategorileri

- **Korea**
- **Japan**
- **China**
- **Menü** (Tüm yemekleri gösterir)
