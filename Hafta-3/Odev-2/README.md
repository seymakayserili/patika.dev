#  Instagram Clone (Bootstrap 4.5.3)

Bu proje, **Bootstrap 4.5.3** ve **Font Awesome 7** kullanılarak hazırlanmış bir **Instagram arayüz klonudur**.  
Proje, responsive (duyarlı) bir tasarıma sahiptir ve Instagram'ın temel görünümünü (sol menü, gönderiler, sağ öneri paneli) taklit eder.

---

## Özellikler

- **Bootstrap 4.5.3** altyapısı ile responsive tasarım  
- **Font Awesome** ikon kütüphanesi kullanımı  
- Sol menüde navigasyon bağlantıları (Anasayfa, Ara, Keşfet, Reels, Mesajlar vb.)  
- Orta kısımda gönderi (post) akışı ve hikayeler (story) alanı  
- Sağ panelde profil bölümü ve “Senin için öneriler” alanı  
- Örnek içerikler: `picsum.photos` görselleriyle dinamik placeholder görüntüler  
- Modern ve sade arayüz  

---

##  Kullanılan Teknolojiler

| Teknoloji | Açıklama |
|------------|-----------|
| **HTML5** | Sayfa yapısı |
| **CSS3** | Temel stiller ve görsel düzen |
| **Bootstrap 4.5.3** | Grid sistemi, layout ve responsive yapı |
| **Font Awesome 7.0.1** | İkon seti |
| **JavaScript (Bootstrap Bundle)** | Bootstrap bileşenleri için etkileşim |
